5-12-2025: Added Export to CSV and Excel
The action buttons for exporting to excel and csv are now live!
It will export as result.csv/xlsx and only give you the filtered results and choosen columns by the user.

5-12-2025: Added count to email recipients in email confirmation
We now have a total recipients item in the email confirmation page for sending emails. This will make it easier to confirm the number of emails being sent out prior to sending.

5-12-2025: Created a Dynamic Details Page
We now have a dynamic details page that allows you to see the details of the selected person. It does have logic started for locations but is not fully functional with data yet.

We now have a search input bar that allows you to search for filters by name. This doesn't search the filter tree names but the values available in the filter tree groups.

We now have a column chooser modal that allows you to select which columns you want to see in the results.

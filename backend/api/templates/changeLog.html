Change Log

5-22-2025: A long list of GUI changes

  1. Added Crypta 2.0 to Header.
  2. Changed home page to database and what was the home page to change log.
  3. Updated URL paths.
  4. Made the relationship button on details page a teams message link.
  5. Update database table with pagination.
  6. Update database with row moving capabilities.
  7. Rounded table results for future main usage with new database look upcoming.
  8. Matched Gradient to Workday for details page.
  9. Updated rounding to better match workday.
  10. Update primary info with boxes that Bill Weldon prefered.
  11. Added Main overflow feature for horizontal and vertical main section of details page.
  12. Added sticky feature to header
  13. Added sticky feature to left nav of details page.
  14. Updated return button to match action button.
  15. Fixed aspect ratio on photos.
  16. Shrunk left nav using css.
  17. Added label for which database you query.
  18. Added new and improved stat filtering.
  19. Added new icons for default usage on details pages.
5-12-2025: Created a search feature for the application
We can now use the search bar to find detail pages from the crypta database and pull specific detail pages up.

5-12-2025: Added dynamic address query system with google for physical locations on Details Page.
Created a basic django parser on the page to make a query with google based on the physical location of churches and schools.

5-12-2025: Added locations functionality for basic location data on Details Page.
Went through and added logic for when location id's are passed to the details Page.

5-12-2025: Added Export to CSV and Excel
The action buttons for exporting to excel and csv are now live!
It will export as result.csv/xlsx and only give you the filtered results and choosen columns by the user.

5-12-2025: Added count to email recipients in email confirmation
We now have a total recipients item in the email confirmation page for sending emails. This will make it easier to confirm the number of emails being sent out prior to sending.

5-12-2025: Created a Dynamic Details Page
We now have a dynamic details page that allows you to see the details of the selected person. It does have logic started for locations but is not fully functional with data yet.

5-12-2025: Added Search Filters
We now have a search input bar that allows you to search for filters by name. This doesn't search the filter tree names but the values available in the filter tree groups.

5-10-2025: Added Column Chooser
We now have a column chooser modal that allows you to select which columns you want to see in the results.

5-8-2025: Header & GUI Updates
We have updated the header to be more user friendly and match more with workday example provided by Adam Padron.

5-5-2025: Introduced Action Buttons
You'll find place holders for exporting to excel, csv, and mailing labels.
But the Send email option is under live development and already has a working attachment system and body expansion setup.

4-25-2025: Entire Filtering & Sorting System Operational
You can find the enhanced filtering system here: Links
You'll find it more user friendly and similar to Amazon purchasing experiences.
